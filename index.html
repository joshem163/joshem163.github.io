Research

I specialize in topological machine learning on graphs, particularly power grid networks. My research includes:

  • Topological graph neural networks (Wise-GNN and ClassContrast models)
  • Power grid resiliency improvement
  • Cyberattack detection using topological data analysis
  • Ongoing work in temporal graph representation learning for power grid outage prediction

Publications

  • Wise-GNN: Topological Graph Neural Networks for Power Grid Networks - under review at ICLR 2025
  • ClassContrast: Contrastive Learning for Node Classification - under review at AAAI 2025
  • Power Grid Resiliency Improvement with Topological Machine Learning - Published in XYZ Journal
